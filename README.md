# Unitedq Uarries static site

Static GitHub Pages export for the SANCO Group mirrored site.

## Structure

- `index.html` - homepage served by GitHub Pages
- `index.php/` - static page paths exported from the original WordPress site
- `wp-content/` and `wp-includes/` - static assets required by the exported pages
- `robots.txt`, `404.html`, `.nojekyll` - GitHub Pages support files
